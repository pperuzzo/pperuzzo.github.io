overview

in 2020, it became particularly difficult for interior designers to collaborate. by leveraging vr technology, i created a collaborative (multiplayer) software for designers to collaborate in a virtual space.

i started by fully developing the vr client software from scratch using unity and c#. by leveraging my expertise in writing c# unity code i ensured that ambient software was well-structured, easy to maintain, performant, modular, and resuable.

then, to implement the real-time vr multiplayer part of the software, i choose photon engine.
this allowed users to interact with each other within the virtual environment and communicate through a voip system.
the multiplayer part consists of 2 key components:
- custom web server with a database
- custom photon server

for the web server, i deployed a php laravel server on aws elastic beanstalk. the key role of the server is to listen to photon's web hook events and update the aws rds database accordingly.
the custom web server facilitated seamless communication between the clients (unity) application, allowing for efficient data management and processing.

the custom photon server was needed to allow customized behaviour in the photon event handling system.
firsly, i edited the open source c# photon server code by introducing the needed events. then i deployed the photon server on a windows machine on aws ec2.

finally, with the 3 components fully deployed (client application, server, photon server), the project was completed.
